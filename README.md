
## 一起中抽奖小程序--到店引流工具

### 切合人群属性
##### 公域用户、小区用户、私域用户
### 切合门店属性
##### 商品品类多、消费频次略高、有滞销（过期）、产品有限制（年龄）,门店举例：水果店、便利店、母婴店、奶站
### 抽奖有哪些好处
##### 抽奖活动可以吸引更多用户关注和参与，用户可以将抽奖活动分享给好友，增加社交互动；<br>通过抽奖活动，商家可以收集到更多关于用户的数据，以便更好地了解用户需求和喜好。<br>用户中奖后需要到店领取奖品进而可以刺激用户购买更多的产品或服务，从而增加销售额
